# SocialMedia

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.7.

## Development backend

Run `npm run dev-server`. It uses port `3000`.

## Development frontend

Run `ng serve`. It automatically navigates to `http://localhost:4200`.

## Development server

Run `npm run dev`. It runs dev backend and dev frontend (both previous commands).

## Production server

Run `npm run prod`. It builds frontend and runs backend.
