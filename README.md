# Responsive SocialMedia web - inspired on Twitter

Live Demo: http://social-media.tomastm.me/

In this Social Media web you can:
- Post, comment, like, share and delete posts
- Posts support multiple image upload as well
- Follow and unfollow users
- View all posts or only from those who you follow
- Edit your profile
- Receive notifications
- Manage your opened sessions and revoke its access

The project was built with the MEAN Stack: MongoDB, ExpressJS, Angular, NodeJS.

Main characteristics:
- **Lazy loaded** modules
- Use of **RxJS**
- Use of **Factories** in user profile tooltip (when you hover your mouse over an avatar or username)
- Use of **Views** in modals (for multiple concatenated modals)
- JsonWebToken (**JWT**) authentication
- Infinite post threads (like Twitter) but implemented with MongoDB
- **Error handling** on both front end and back end
